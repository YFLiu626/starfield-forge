Install:

1. Extract the whole folder.
2. Double-click install.bat or the Chinese install batch file.
3. Choose the install location when prompted. The app will be installed into a "3D Starfield Generator" folder inside the selected location.
4. Launch from the desktop shortcut or Start Menu shortcut named "3D Starfield Generator".
5. Uninstall from Windows Installed Apps, or run uninstall.bat from the install folder.

Portable:

You can also run the app exe directly from this folder. Keep all files and folders together.

If Windows shows an unknown publisher warning, choose "More info" and continue. This build is not code signed.
